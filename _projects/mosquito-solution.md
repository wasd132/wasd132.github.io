## 사업 개요

세종시·지자체와 협력하여, 태양광·AI·IoT 기반 모기 퇴치기/감지기 시제품을 개발·설치하고, 실시간 모니터링 및 방역 효율화 성과를 달성한 리빙랩 프로젝트입니다.

## 주요 내용

| 구분 | 내용 |
|------|------|
| **시제품** | 모기 퇴치기 5대, 감지기 5대 제작 및 교내·공공장소 설치 |
| **AI 모기 인식** | YOLOv5 기반 이미지 인식 S/W 개발, 정확도 82% 달성 |
| **태양광 전원** | 12hr 배터리 전환, BMS·PCB 커스텀 설계 |
| **실시간 모니터링** | 카메라·센서 데이터 수집, 대시보드 시각화, 임계값 초과 알림 |
| **지자체 협력** | 세종시 질병관리청 등 3개 이상 기관 협력, 현장 자문·테스트 |

## 시스템 구조

```
태양광 패널 → 배터리/BMS/PCB → 퇴치기/감지기(라즈베리파이, 카메라, 팬, UV LED)
    ├─ AI 모기 인식(YOLOv5) → 실시간 모니터링/알림
    └─ 데이터 → 대시보드/지자체 협력기관
```

## 담당 업무 (팀장/AI·S/W 총괄)

- 사업 기획·지자체 협력(세종시 질병관리청 등) 및 일정 총괄
- YOLOv5 기반 모기 인식 S/W 개발, 학습데이터 수집·라벨링·정확도 개선
- 퇴치기/감지기 시제품 제작(라즈베리파이, 카메라, 팬, UV LED, PCB)
- 실시간 모니터링 대시보드 설계 및 임계값 알림 시스템 구축
- 다학제(전자/컴공/경영) 팀 협업 및 현장 테스트 주도

## 기술 스택

| 기술 | 역할 |
|------|------|
| **YOLOv5** | 모기 인식 커스텀 모델, 이미지 S/W 개발 |
| **Python + OpenCV** | 실시간 영상 캡처, 추론 파이프라인 |
| **Raspberry Pi** | 엣지 디바이스, 팬/UV LED/카메라 제어 |
| **PCB/BMS** | 배터리 관리, 전원 회로 설계 |
| **Dashboard(웹)** | 데이터 시각화, 임계값 알림 |
