## 프로젝트 배경

매년 여름 세종시를 포함한 중부 지역에서 말라리아 매개 모기가 대량 발생하여 시민 건강을 위협합니다. 기존 방제 방식은 광범위한 살충제 살포에 의존하여 환경 문제가 발생합니다. 이를 해결하기 위해 AI 기반의 정밀 해충 탐지 시스템을 설계했습니다.

## 주요 기능

- **해충 실시간 탐지**: YOLOv5 모델을 Raspberry PI에 배포하여 모기 발생 지역을 실시간 모니터링
- **태양광 전력 공급**: 외부 전원 없이 독립적으로 운영 가능한 태양광 기반 전력 시스템
- **발생 알림 시스템**: 특정 임계값 이상 모기 탐지 시 관리 서버로 알림 전송

## 기술 스택 상세

### Raspberry PI
- 엣지 디바이스로서 현장에 배치하여 실시간 추론 수행
- 카메라 모듈과 연동하여 영상 스트리밍 및 분석

### YOLOv5 경량화
- Raspberry PI 환경에 맞춰 모델 경량화(Pruning, Quantization)
- 추론 속도와 정확도 간 균형 최적화

## 시스템 아키텍처

1. 태양광 패널 → 배터리 충전 → Raspberry PI 전력 공급
2. 카메라 모듈 → 실시간 영상 캡처
3. YOLOv5 추론 → 해충 탐지/카운팅
4. 임계값 초과 시 → 서버 알림 전송
